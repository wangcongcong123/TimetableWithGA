# CourseTableWithGA（使用遗传算法自动安排课表）

**项目总结**



拓展优势：

1.引入组的概念(可以当作学期数用也可以用于同学分组等等)

2.输出结果以图形化页面，多少个组将会有多少个课程表生成

3.拓展支持导出课表功能



不足劣势：

1. 配置为hardcode模式，暂且支持非运行时配置
2. 课表图形界面目前仅限于默认时间段配置，但任意配置都满足命令行输出的正确性。
